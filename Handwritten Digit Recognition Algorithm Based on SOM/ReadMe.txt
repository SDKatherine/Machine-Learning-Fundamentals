本算法是由Python3实现，基于自组织映射的手写数字识别算法，文件夹中包含如下内容：
1. SOM_MNIST_data.txt是数据集。为了保证程序的正常运行，数据集必须与代码放置在同一个文件夹下，否则需要更改代码。
2. SOM.ipynb是Jupyter Notebook下的代码格式，可以直接在Jupyter下运行。
3. SOM.py是Python3代码，可以使用任何支持Python3的编辑器编译。
4. 三个.txt文件是运行代码的中间产物，final.txt中存储的是最终的权重值，initial.txt是初始时的权重值，train.txt是每次迭代的mse。



This algorithm is implemented by Python3 and is a handwritten digit recognition algorithm based on Self-Organizing Mapping. The folder contains the following contents:
1. SOM_MNIST_data.txt is the dataset. In order to ensure the normal operation of the program, the dataset must be placed in the same folder as the code, otherwise the code needs to be changed.
2. SOM.ipynb is the code format under Jupyter Notebook and can be run directly under Jupyter.
3. SOM.py is Python3 code and can be compiled with any editor that supports Python3.
4. Three .txt files are the intermediate products of the code, final.txt stores the final weight value, initial.txt is the initial weight value, and train.txt is the mse of each iteration.
